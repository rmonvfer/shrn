Introducción

Shrn es una plataforma que conecta a investigadores de ciberseguridad IoT con profesionales que trabajan asegurando este tipo de sistemas (pentesters, bug bounty hackers...).

Los usuarios deberán registrarse para poder hacer uso de la plataforma, ya que todas las acciones están asociadas a una cuenta de usuario. Algunas son:

  • Creación de consultas (posts)
  • Comentario de consultas publicadas
  • Votación de comentarios y consultas
  • Importación y exportación de consultas y comentarios

No es necesaria una clave de API para llevar a cabo estas acciones porque Shrn genera una automáticamente mediante un servicio habilitado por Shodan.

Operaciones básicas

Las operaciones que se describen a continuación constituyen el núcleo de la interacción del usuario con la plataforma, de manera que este tendría una experiencia completa limitándose a usar tan solo estas.

Se supone que el usuario ya dispone de una cuenta y que ha iniciado sesión. Puede encontrarse más información al respecto en la página de inicio

Creación de consultas

Para crear una consulta el usuario deberá dirigirse a la página de creación de consultas seleccionando el apartado correspondiente en la barra de navegación superior y posteriormente haciendo clic en la opción de "Nueva Consulta". Alternativamente puede acceder mediante este enlace

El usuario podrá ver el formulario de creación de consulta que deberá rellenar con los datos que se indican en el mismo:

  • Título de la consulta
  • Descripción
  • Contenido (la consulta en sí)

Finalmente, si no aparece ningún error, la consulta estaría creada y el usuario sería redirigido a la sección de consultas publicadas.

Publicación de comentarios

Desde la sección principal ("Comunidad"), el usuario verá una lista de consultas publicadas por otros usuarios (no aparece la suya propia), al seleccionar una de ellas haciendo clic en la misma será redirigido a una sección de detalle de la consulta.

Una vez allí, haciendo uso del botón "Mostrar comentarios" el usuario podrá ver un listado de los comentarios ya publicados en la consulta. Para enviar un comentario tan solo debe rellenar la caja de texto y pulsar "Enviar"

Funcionalidad avanzada

Shrn soporta una serie de funciones diseñadas para usuarios con la necesidad de guardar una copia de seguridad de sus consultas y/o comentarios. En concreto:

Importado

De documentos SQSF y XML. La importación restaura las consultas y comentarios publicados por el usuario propietario del documento exportado (comprobado en el momento de la importación)

Conversión

En el momento de la importación es posible generar un archivo JSON para emplear los datos de la consulta en otras herramientas.

Exportado

A formato SQSF, basado en XML y de uso interno para gestionar el procedimiento de copia de seguridad y restauración de consultas y comentarios.
