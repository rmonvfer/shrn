Preguntas frecuentes

¿Qué es shodan?

Shodan es un buscador especializado en dispositivos conectados (también denominados IoT), bases de datos, servidores... Shodan busca de manera constante nuevos dispositivos en Internet y los indexa para que puedas buscarlos,

¿No necesito una API Key?

En teoría si, pero Shodan permite generar claves de API para pruebas y eso es justo lo que hace esta web, cuando creas una cuenta se asigna automáticamente una de estas claves para que puedas ejecutar las consultas compartidas por el resto de miembros de la comunidad.

¿Que más puedo hacer?

Hemos creado una sección específica para responder a esa pregunta, haz clic aquí para ver una descripción más detallada de todas las funcionalidades de Shrn.

En resumen, puedes publicar y leer consultas (usando el lenguaje específico de consulta de Shodan), publicar comentarios, ejecutar consultas en directo e incluso puedes filtrar los resultados de una consulta concreta según tu localización.

Existen otras funcionalidades avanzadas como la importación/exportación a SQSF (un formato especial basado en XML que te permite guardar una copia de seguridad de tus consultas y comentarios publicados) y la conversión a JSON, otro formato muy popular de intercambio de datos.

¿Cómo puedo ponerme en contacto contigo?

Ahora mismo esto es un proyecto sin ninguna pretensión más allá de aprobar una asignatura, si tienes curiosidad por saber más acerca de este sitio web puedes enviarme un email, ¡aunque no prometo contestar!
